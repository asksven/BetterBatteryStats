BetterBatteryStatsRelease Notes

v1.6.1.0 2012-02-10

  • Fixed graphs on ICS
  • Moved to hollow style, increased spinner height for better usability on ICS
  • Fixed bug causing "since charged" reference to be called continuously after reaching 100%
  • Removed pref to hide "since..." from display to make screenshots more readable
  • Removed Service due to alternate design of handling the battery charged event
  • Removed auto-refresh to improve overall performance
  • Alarms with references and moved to main view (!!!be aware that this causes more frequent calls to 'su' as alarm dump requires root access)
  • Added follow-me and rate buttons to readme
  • Battery Service proxy is now a singleton for a better stability on MIUI
  • Click on Alarms / Kernel wakelocks shows details dialog
